## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```
## ENV
```bash
# Port
PORT = 'Your Port'
# mongodbUrl
mongodbUrl = 'Your Url'
```
## Development Server

Start the development server on `http://localhost:{{PORT}}`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```
